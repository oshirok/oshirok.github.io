Personal Projects

These are my favorite personal projects
that I would like to showcase to you

  • BusWasabi

    Shows the live locations of King County busses on a Map. Interpolates the schedule data and the last known location of the user-selected bus to mark the approximate location of the bus on a map. Written using Javascript and Leaflet.js

  • WordNoodle

    Word-Search like Android game designed to help young students memorize groups of names such as the names of the 50 states. It is solely written using Java and LibGDX and is available on Google Play.

  • Google Drive Parallel Uploader

    Mirrors the contents of a local folder to a Google Drive folder without uploading duplicates. Enables parallel uploading at up to 14.6MB/s compared to the Google's browser client's 4.5MB/s maximum upload speed. Written using the Google Drive API, Java and the Fork/Join Framework

  • VertaHub

    WebApp designed to centralize and facilitate interdepartmental communcation within Vertafore. Vertaforians can post Event's, Lunches and Questions and respond to other posts via comments. Created with Node.js and the MEAN stack.

  • BitWiser

    A Web Game designed to help CS students practice bitwise operations through solving progressively harder bit operation problems. Created as a team during the 2014 DubHacks Hackathon using Node.js and the HackerRank API.

  • UW Map

    An animated map of UW containing Points of Interest and calculates directions between POIs using Dijkstra's Algorithm. Created as a final project for CSE 331 using Java and the JSwing Library.
